sed用法总结

1. 简介

sed是一种流编辑器，非交互式。sed将要处理的文件以行为单位读入；将此行放入临时缓冲区；在缓冲区对流副本进行编辑；编辑后将结果送到标注输出。

2. 使用方法

首先定位要处理的行（定址），再操作（命令，选项），遵照正则表达式规则模糊搜索

sed [address[,address]][!]{cmd}

  1. 定址

    sed选定要处理的行，由数字，正则，或二者结合确定。默认全部行
